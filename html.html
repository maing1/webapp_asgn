Hypertext Markup Language (HTML)

HTML is the standard language used to create web pages.

Features

  • Uses tags to structure content
  • Supports text, images, and links

Example

HTML is used to create headings, paragraphs, and images on a webpage.

Importance

It forms the foundation of all websites.
