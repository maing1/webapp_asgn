Uniform Resource Locator (URL)

A URL is an identifier for the location of a document on the internet.

Structure of a URL

  • Protocol (https://)
  • Domain name (example.com)
  • TLD (.com)
  • Path (/page)

Example

https://www.google.com

Importance

URLs help users locate and access specific web pages easily.
